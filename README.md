🚀 AI Virtual Development Pod

An AI-powered multi-agent system that simulates a complete software development team using locally hosted Large Language Models. The system automates project planning, code generation, quality analysis, and evaluation from a simple user prompt.

📌 Overview

The AI Virtual Development Pod transforms a user’s project idea into structured development outputs using multiple AI agents. It mimics a real-world development workflow by assigning different responsibilities to specialized agents such as Project Manager, Developer, QA, and Evaluator.

The system uses Ollama to run Llama 3 locally, ensuring privacy, offline capability, and zero API cost.
